# UPC-Animations-and-Graphics
## A Collection of Animations, Graphics, and other teaching/visualization tools to show off Ultra-Peripherial Collisions

### Most of these animations are made using python and the [manim](https://www.manim.community/) library
 - Installation and Documentation for manim [here](https://docs.manim.community/en/stable/tutorials/quickstart.html)

### Files include:
- UPC Pb-Pb interaction to make J/psi into dimuon (`upc_animation.py`)
